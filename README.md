# 📈 한국 주식 뉴스·트렌드 분석 대시보드

매일 관심 종목의 **시세 지표 + 최신 뉴스**를 모아서, 주목도 점수와 그 **근거**를
정리해 웹 대시보드로 보여주는 프로그램입니다. (프로그래밍 입문자용)

> ⚠️ **중요:** 이 도구는 정보를 정리해 판단을 돕는 보조 도구일 뿐 **투자 자문이 아닙니다.**
> 점수와 근거는 수익을 보장하지 않으며, 모든 투자 결정과 책임은 본인에게 있습니다.

---

## 🚀 실행하는 법 — 파일 하나만 더블클릭하세요!

복잡한 명령어 입력은 필요 없습니다. **본인 컴퓨터에 맞는 시작 파일을 더블클릭**하면,
파이썬 확인 → 필요한 프로그램 설치 → 대시보드 실행까지 자동으로 진행됩니다.

| 내 컴퓨터 | 더블클릭할 파일 |
|-----------|----------------|
| 🪟 **윈도우** | `시작하기_윈도우.bat` |
| 🍎 **맥** | `시작하기_맥.command` |

> 다른 `.py` 파일들은 **열 필요 없습니다.** 위 시작 파일 하나만 누르면 됩니다.

잠시 후 웹브라우저에 대시보드가 자동으로 열립니다.
왼쪽의 **'오늘 데이터 분석하기'** 버튼을 누르면 분석이 시작됩니다.
프로그램을 끄려면 같이 떠 있는 검은 창(터미널)을 닫으면 됩니다.

### 🪟 윈도우에서 처음 실행 시
- 파이썬이 없으면 자동으로 설치 페이지가 열립니다. 설치할 때
  **"Add Python to PATH"** 체크박스를 꼭 켠 뒤, 시작 파일을 다시 더블클릭하세요.
- "Windows의 PC 보호" 경고가 뜨면 **추가 정보 → 실행**을 누르세요.

### 🍎 맥에서 처음 실행 시
- 처음에는 더블클릭 대신 파일을 **마우스 오른쪽 클릭 → 열기**로 실행하세요
  (보안 경고를 한 번만 통과하면 됩니다).
- "권한이 없다"는 메시지가 나오면, 터미널에서 폴더로 이동한 뒤
  `chmod +x 시작하기_맥.command` 를 한 번 실행하세요.

## 🤖 (선택) Claude AI 근거 설명 켜기

API 키 없이도 **규칙 기반**으로 잘 작동합니다.
뉴스 맥락까지 읽는 더 똑똑한 근거 설명을 원하면 Anthropic API 키를 설정하세요.

- Windows: `set ANTHROPIC_API_KEY=여기에_키`
- Mac/Linux: `export ANTHROPIC_API_KEY=여기에_키`

키를 설정한 뒤 다시 `streamlit run app.py` 를 실행하면 자동으로 AI 모드가 켜집니다.
(API 사용 시 소액의 요금이 발생합니다.)

## 📂 파일 구조 (각 파일이 하는 일)

| 파일 | 역할 |
|------|------|
| **`시작하기_윈도우.bat`** | 🪟 윈도우용 — **이걸 더블클릭하면 됩니다** |
| **`시작하기_맥.command`** | 🍎 맥용 — **이걸 더블클릭하면 됩니다** |
| `app.py` | 웹 대시보드 (시작 파일이 자동으로 실행) |
| `config.py` | 관심종목·설정 (여기서 종목을 바꾸세요) |
| `data_collector.py` | 시세 수집 + 지표 계산 (이동평균, RSI, 거래량 등) |
| `news_collector.py` | 네이버 뉴스 수집 |
| `analyzer.py` | 점수화 + 추천 근거 생성 (핵심) |
| `storage.py` | 추천 기록 저장 (나중에 검증용) |

---

## 🔧 자주 손볼 곳

- **종목 바꾸기:** `config.py` 의 `WATCHLIST` 수정
- **추천 개수 바꾸기:** `config.py` 의 `TOP_N` 수정
